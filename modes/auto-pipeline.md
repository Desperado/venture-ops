# Auto-Pipeline Mode

Use when the user pastes an investor, accelerator, grant, studio, angel, or program URL/description without a narrower command.

Steps:
1. Load shared context and user-layer source files.
2. If the target is time-sensitive or web-specific, browse and verify current facts.
3. Evaluate fit using the shared rubric.
4. Create a report in `reports/{target-slug}-{YYYY-MM-DD}.md`.
5. Recommend one of: priority, pursue, conditional, monitor, skip.
6. If fit is 4.0+, propose tailored deck, memo, or outreach changes.
7. Provide a tracker-ready row. Do not claim the founder applied or contacted anyone.
