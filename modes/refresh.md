# Refresh Mode

Refresh Venture-Ops from the latest available materials:
- existing pitch deck
- startup front page
- current traction notes
- founder answers to targeted questions

Workflow:
1. Read the current deck or memo if provided.
2. Inspect the startup front page and product messaging.
3. Identify stale claims, missing metrics, missing customer proof, outdated screenshots, or inconsistent narrative.
4. Ask the founder a short list of knowledgeable questions that would materially improve the deck.
5. Update the relevant user-layer files with the new facts once confirmed.
6. Produce a heads-up list of slides or sections that should be refreshed now.
