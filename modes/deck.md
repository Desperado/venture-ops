# Deck Mode

Generate fundraising materials for a specific target.

Inputs:
- `startup.md`
- `founder-bio.md`
- `traction-digest.md`
- target evaluation or report when available
- `templates/deck-template.html` when producing HTML or PDF

Default package:
1. 10-slide pitch deck outline
2. target-specific narrative angle
3. proof gaps and suggested metrics or screenshots
4. stale or missing slide warnings
5. investor memo summary
6. optional HTML deck using `templates/deck-template.html`

Compare the requested material against any existing deck plus the startup front page. If claims, screenshots, traction, team, or product story look stale, call it out clearly and ask a short set of high-value founder questions.

Keep claims specific and evidence-backed. Mark assumptions clearly.
