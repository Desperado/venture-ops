# Shared Venture-Ops Context

Load before most modes.

## Source Files

Read these when present:
- `startup.md` for company, product, market, and traction
- `founder-bio.md` for founder credibility
- `traction-digest.md` for compact proof points
- `market-watch.md` for recent trends, news, and timing hooks
- `config/profile.yml` for raise and targeting filters
- `modes/_profile.md` for founder-specific preferences and scoring adjustments
- `investors.yml` for known targets

## Scoring Rubric

Score each target from 1.0 to 5.0:

| Dimension | Weight | What to assess |
| --- | ---: | --- |
| Stage fit | 15% | target invests at this round and company maturity |
| Thesis fit | 20% | sector, business model, product category, market timing |
| Geography/legal fit | 10% | incorporation, founder location, customer market |
| Check/program fit | 10% | check size, ownership, cohort terms, round construction |
| Portfolio signal | 10% | relevant portfolio, conflicts, pattern of similar bets |
| Proof readiness | 15% | traction, demo, metrics, customer evidence, founder-market fit |
| Access path | 10% | warm intro, application route, partner relevance |
| Strategic value | 10% | brand, network, customers, follow-on, hiring, operator support |

Recommendation bands:
- 4.5-5.0: priority target, prepare highly tailored package
- 4.0-4.4: strong fit, pursue if access path exists
- 3.5-3.9: conditional, improve proof or target a specific partner/program
- 3.0-3.4: monitor or deprioritize
- below 3.0: skip

## Output Contract

For evaluations, produce:
1. Target summary
2. Fit score and recommendation
3. Why this target might say yes
4. Likely objections
5. Best approach path
6. Tailored deck or memo changes
7. Missing proof or stale-slide warnings
8. Short founder questions if current facts are unclear
9. Application or outreach draft if requested
10. Tracker-ready row

## Current Facts

Investor facts and market narrative change. Browse when answering about current partner rosters, deadlines, program terms, application windows, check sizes, thesis pages, recent investments, startup front-page claims, sector news, competitor moves, or trends.

## Human-in-the-Loop Rule

Draft, evaluate, and prepare. Never submit applications, send outreach, or represent that contact has happened unless the user explicitly tells you they did it.
